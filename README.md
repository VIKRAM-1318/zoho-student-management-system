Zoho Creator – Student Management System
Overview

A student management application built using Zoho Creator to manage student records, faculty details, and attendance tracking with percentage analytics.

Features

Student Registration Form

Faculty Management

Attendance Tracking System

Automated Attendance Percentage Calculation

Pivot Attendance Report

Role-Based Access Control

Deployed using Zoho Creator Environment (Development → Stage)

Tech Stack

Zoho Creator

Deluge Scripting

Pivot Reports & Aggregate Formulas

Deployment

Hosted on Zoho Creator Platform
Versioned release (v1.0) published to Stage Environment.
